### node-cmd
---

Command line tool that aggregates information from the Bitcoin RPC.

#### Setup
```
    git clone https://github.com/ns-xvrn/node-cmd.git &&
    cd node-cmd &&
    pip install -r requirements.txt
```
---

#### info view
`python node_cmd.py info` or `python node_cmd.py`
